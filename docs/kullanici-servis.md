# kullanici-servis

Quarkus based service for managing users.

## Endpoints

| Method | Path | Description |
|-------|------|-------------|
| GET | `/kullanici` | List all users. |
| POST | `/kullanici` | Create a new user. |
| PUT | `/kullanici/{id}` | Update an existing user. |
| DELETE | `/kullanici/{id}` | Delete a user. |
